Description

This project is based on Hospital Management System data interpretation using Excel. It analyzes hospital data like patients, doctors, treatments, and billing to generate useful insights.

Features:-
Patient data analysis
Doctor and department records
Treatment and diagnosis tracking
Billing and expense analysis
Charts and graphs for visualization

Tools Used:-
Microsoft Excel
Data Analysis (Sorting, Filtering, Pivot Tables)
Charts (Bar, Pie, Line)

How to Use:-
Open the Excel file
Navigate through different sheets
Use filters and pivot tables to analyze data
View charts for better understanding

 File Included:-
Excel file (.xlsx) containing hospital data and analysis

 Objective:-

To understand and analyze hospital data for better decision-making and management.
